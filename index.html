01  為什麼需要「高階」銅箔？— 趨膚效應
高頻高速傳輸下，電子訊號的物理限制決定銅箔品質要求
🌟 小學生版本（非專業也能懂）
銅箔拿來做什麼？
📱 手機天線 & 電路板   🔋 電動車電池   🤖 AI 伺服器
趨膚效應是什麼？
就像「魚只在水面游泳」，高頻訊號不走導線中心，只貼著最外層跑。
→ 表面平整度比厚度重要 100 倍！
Standard / VLP / HVLP？
Standard 🪨 = 石子路（坑坑洞洞）
VLP 🚗 = 柏油路（比較平）
HVLP 🏎 = F1 賽道（極平滑）
路越平，訊號車跑越快！
[業務語言]  山脈 vs 高速公路
訊號只走「最外層表面」跑
▸ 高粗糙度銅箔 → 表面是「連綿高山」→ 訊號嚴重失真
▸ VLP / HVLP：把山脈夷平 → 打造高速公路！
VLP = Very Low Profile（超低粗糙度）
HVLP = Hyper Very Low Profile（極限低粗糙度）
等級Rz (μm)應用訊號損耗
Standard> 3.0一般 PCB高 ▲
VLP< 2.05G / mmWave中 →
HVLP< 1.0AI 伺服器 / 77GHz低 ▼
HVLP+< 0.5先進封裝 / 毫米波極低 ▼▼
目前業界前沿：HVLP Rz < 0.5 μm（先進晶片封裝要求）
[科學底層]  Skin Effect 趨膚效應
趨膚深度公式：
δ = 1 / √( π · f · μ · σ )
δ：趨膚深度 | f：頻率 | μ：導磁率 | σ：導電率
頻率 f趨膚深度 δ銅箔等級
1 GHz~ 2.1 μmVLP
10 GHz~ 0.7 μmHVLP
28 GHz~ 0.4 μmHVLP+
77 GHz~ 0.2 μmHVLP+
頻率越高 → 電流越集中在表面薄層 → 導體內部完全無效
趨膚深度 δ 必須 > 表面粗糙度 Rz，訊號才能正常傳導
⚠ 物理矛盾臨界點
當 δ < Rz 時， 訊號已無法貼著「平面」傳導——
電流將在銅箔表面的微觀山脈中 漫射耗散，
此段損耗屬熱力學不可逆損失，無法以等化器補償。
例：28 GHz 下 δ ≈ 0.38 μm；Standard 銅箔 Rz > 3 μm → 訊號深陷山谷，損耗超出規格 8× 以上
關鍵結論：高頻時代，銅箔「表面品質」決定一切；HVLP 全名 Hyper Very Low Profile，Rz < 0.5 μm 是 AI 晶片封裝最新要求
02  推拉效應 × 微觀電鍍：一場毫秒級肉搏戰
Suppressor × Leveler × Accelerator — 點擊左側按鈕觸發各階段沉浸動畫
🔵 Suppressor — PEG（抑制劑）
PEG 長鏈聚合物 + Cl⁻ 協同吸附於銅表面，形成灰色擴散屏障。
全面覆蓋、均勻阻擋 → 降低整體沉積速率。
🟠 Leveler — 膠原蛋白胜肽（整平劑）
帶正電胜肽（NH₃⁺），被電場梯度牽引至山峰頂端。
精準封鎖凸起處，阻止峰頂繼續生長 → 物理整平。
⚡ Accelerator / SPS（加速劑）
SPS 雙硫鍵（S─S）在谷底斷裂，釋放 Cu⁺ 錯合催化位點。
谷底加速 ÷ 峰頂阻斷 = 超等角填孔
⚖️ 動態平衡關鍵：三種添加劑濃度比例決定填孔成敗。
SPS 過多 → 過填（over-fill）凸起；PEG 不足 → 四壁空洞。
▸ 階段一 無添加劑：Cu²⁺ 隨機沉積 — 山峰持續竄高，谷底永遠無法填平
毫秒時間尺度：吸附 ~10 ms · 脫附 ~1 ms · SPS 置換 <1 ms

04  鹼處理 (Type B) vs. 酸處理 (Type A)
硫酸銅電解液 pH ≈ 1  │  pH < pI → 蛋白質帶正電 → 吸附Cu表面  │  pI 精控決定銅箔品質
關鍵：牛皮因高交聯度採 Type B 鹼處理；豬骨/牛骨同理。只有 Type B 才能實現 VLP/HVLP 所需的精準 pI 設計。

06  研發主管沒注意到的三個「技術盲區」
這些盲區，正是切入差異化銷售的黃金切入點
盲區 01 — 被 Mw 欺騙
「我們的明膠平均分子量符合規格」
—— 這是最常見的誤判說詞
PDI = Polydispersity Index
多分散性指數 — 描述分子量分佈的寬窄
PDI = Mw / Mn　→　越接近 1.0 越均一
PDI 發散導致奈米級沉積缺陷：
　大分子堆積 → 局部過抑制 → 黑點
　小分子空洞 → 局部欠覆蓋 → 針孔
正確 KPI：PDI 值，而非單純的 Mw
盲區 02 — 晶界工程被忽視
「添加劑只是把表面弄平」
添加劑的真實影響：
　改變晶面生長方向 (111)→(200)
　打碎柱狀晶，孕育等軸晶
　提升抗拉強度 (TS) 與延展性 (El%)
不談晶界工程，就是在賣半套方案
盲區 03 — 吸脫附的毫秒級肉搏
「正電荷不是越強越好」
胜肽吸附能必須處於臨界點
若吸附太強：
　SPS 雙硫鍵無法在毫秒內置換
　油門與煞車相互鎖死
　填谷機制完全失效
pI 精控（Type B）的真實商業價值
銷售技巧：「請問您目前如何評估整平劑 PDI 對銅箔結晶結構的影響？」— 一個問題打開高層技術對話
07  結語與差異化定位
1
表面 → 結構
高階銅箔添加劑不只改變 Rz，更改變晶粒形態與機械性質（晶界工程）
2
Mw → PDI
正確的品質 KPI 是多分散性指數 PDI，而非平均分子量 Mw
3
靜態 → 動態　（加速劑 SPS 的核心機制）
SPS 的雙硫鍵（−S−S−）驅動的毫秒級吸脫附平衡，負責加速谷底銅沉積，填平微凹坑
4
為什麼膠原蛋白胜肽（Leveler）才是您的銷售重點？
PEG（抑制劑）與 SPS（加速劑）均已商品化且高度標準化 — 差異化幾乎為零。
膠原蛋白胜肽是唯一需要精準設計 PDI、pI、胺基酸序列的三維知識壁壘添加劑：
① PDI → 1.0　均一覆蓋，消除缺陷　②　pI 精控　決定電場吸附強度
③ Gly-Pro-Hyp 序列　原子級鎖定 (111) 晶面　→　等軸晶孕育
① 戰場：趨膚效應 › ② 賽局：推拉平衡 › ③ 武器：PDI ＆ pI 精控 › ④ 降維：晶面工程
